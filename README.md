# Zadbane — strona usług koszenia trawy

Prosta strona statyczna przygotowana dla lokalnej usługi „Zadbane” z Olsztynka.

## Publikacja na GitHub Pages

1. Utwórz na GitHubie nowe publiczne repozytorium, np. `zadbane-strona`.
2. Wgraj do niego całą zawartość paczki i zatwierdź pliki na gałęzi `main`.
3. Wejdź w **Settings → Pages**.
4. W sekcji **Build and deployment** wybierz **Deploy from a branch**.
5. Wybierz gałąź `main`, katalog `/(root)` i kliknij **Save**.

Po chwili GitHub pokaże bezpłatny adres strony. Domena z OVH może zostać podpięta później w tym samym miejscu, w polu **Custom domain**.

## Najważniejsze pliki

- `index.html` — treść strony,
- `app/globals.css` — wygląd strony,
- `public/hero-garden.webp` — główne zdjęcie,
- `public/favicon.svg` — ikona strony.

Numer telefonu jest już ustawiony na `531 257 929`. Przed publikacją możesz zastąpić zdjęcie `public/hero-garden.webp` własną fotografią realizacji — bez zmiany nazwy pliku.
